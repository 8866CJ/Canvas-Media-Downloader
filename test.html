Canvas Media Downloader - Test Page

This page simulates a Canvas page for testing the extension

Installation Instructions

  1. Open Chrome and go to chrome://extensions/
  2. Enable "Developer mode" (toggle in top-right)
  3. Click "Load unpacked"
  4. Select the extension directory
  5. The extension icon should appear in your toolbar

Testing Instructions

  1. After installing the extension, refresh this page
  2. Click the extension icon to open the popup
  3. Check if the media files below are detected
  4. Toggle between automatic and manual modes
  5. In manual mode, click download buttons in the popup

Expected Behavior

  • Automatic Mode (default): Media files should download automatically when detected
  • Manual Mode: Media files should appear in the popup list with download buttons
  • Popup: Should show count of detected media and allow mode toggling
